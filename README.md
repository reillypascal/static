# Static HTML Site Directory

A curated directory of personal sites built with static HTML + CSS.

## Submission Guidelines

To be included in the directory:

- Your site must use static HTML + CSS (some scripting for extra functionality is fine!)
- It must be your personal site or blog
- The site should reflect something about yourself

## How To Submit Your Site

### Option 1: Pull Request

1. Fork this repository
2. Edit `sites.html` and add your site in alphabetical order by name
3. Follow the existing format: `<li><a href="YOUR-URL" rel="noopener">Your Name</a></li>`
4. Submit a pull request

Please add your site in alphabetical order by your name (first name, last name, username..your choice!)

### Option 2: Email

Email me at [hi@zacharykai.net](mailto:hi@zacharykai.net) with:

- Your name (how you'd like to be listed)
- Your site URL
- A brief note about your site
